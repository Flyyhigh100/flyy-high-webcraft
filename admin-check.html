Supabase Admin Check

This page helps you force a complete logout to ensure you get fresh authentication data.

Steps to resolve admin access issues:

  1. Click the "Force Logout" button above
  2. Close this tab completely
  3. Open your application again at http://localhost:5175
  4. Log in with your credentials
  5. Go to your dashboard
  6. Check if the admin button appears

If you still don't see the admin dashboard:

Make sure you've actually executed the SQL in the Supabase dashboard by following these steps:

  1. Go to your Supabase dashboard
  2. Navigate to "SQL Editor" in the left sidebar
  3. Create a new query
  4. Copy and paste the entire content of the admin-setup.sql file
  5. Click "Run" to execute
  6. Then try the Force Logout steps above
