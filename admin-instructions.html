Setting Up Your Admin Access

Overview

You need to properly set up your admin status in the Supabase database for permanent access to the admin dashboard. I've reverted our temporary "force admin" solution and prepared a proper database setup script.

Note: Once you complete these steps, you'll need to log out and log back in for changes to take effect.

Step 1: Access Supabase Dashboard

Go to the Supabase Dashboard and log in to your account.

Select your project for this website.

Step 2: Open SQL Editor

In the left sidebar, click on "SQL Editor".

Then click "New Query" to create a new SQL query.

Step 3: Execute Admin Setup SQL

I've created a SQL file with the commands to properly set you up as an admin. You need to:

  1. Open the make-admin-user.sql file I created
  2. Copy the entire contents of that file
  3. Paste it into the SQL Editor in Supabase
  4. Click "Run" to execute the script

The SQL will:

  • Create the profiles table if it doesn't exist
  • Clear any existing profiles for your user (to avoid duplicates)
  • Insert your profile with admin role
  • Set up the necessary security policies

Step 4: Verify Admin Access

After running the SQL:

  1. Go back to your application
  2. Log out completely
  3. Log back in with your credentials
  4. Navigate to your dashboard
  5. You should now see the "Admin Dashboard" button

Troubleshooting

If after completing all steps you still don't see the admin dashboard:

  1. Check the browser console for any errors (F12 > Console)
  2. Make sure you completely logged out and logged back in
  3. Try clearing your browser cache and cookies
  4. Verify the SQL executed successfully in Supabase
